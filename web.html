Web design & development

I have created websites that are visually appealing and easy to use. Using HTML, CSS and PHP, I have built solutions that work seamlessly on both desktop and mobile devices. My goal is always to deliver a smooth and clear user experience.

In a collaborative group project, we developed 'Vinemo,' a social platform for wine enthusiasts to share reviews and ratings. My contributions included designing the 'About Us' page and implementing a newsletter subscription feature with database integration. The project utilized tools like HTML5, CSS3, JavaScript, PHP, and MySQL to create an interactive and user-friendly experience. Through teamwork, I honed my design and problem-solving skills, ensuring cohesive visual elements and functional user interactions.
